# 1 vs 1 Mini Games

1台のスマートフォンやタブレットを使って、2人で対戦できるミニゲーム集です。
向かい合って座り、画面の両端を使ってプレイします。

## 収録ゲーム

現在、以下の5つのゲームが収録されています。

### 1. Reflex Battle (リフレックス・バトル)
**早押し対決**
画面の色が変わった瞬間にボタンを押すだけのシンプルな反射神経ゲーム。お手つきに注意！

### 2. Number Duel (ナンバー・デュエル)
**大小判定早押し**
画面に表示された2つの数字のうち、「大きい方」のプレイヤーがボタンを押します。
例: 自分「8」 相手「3」 → 自分が押す。
瞬時の判断力が試されます。3点先取で勝利！

### 3. Pattern Memory (パターン・メモリー)
**記憶力対決**
3x3のグリッドが光る順番を覚え、相手より正確に再現します。
ラウンドが進むにつれてパターンが長くなります。

### 4. Tap Rush (タップ・ラッシュ)
**連打バトル**
5秒間ひたすらボタンを連打！
指の体力が試されるパワー勝負です。

### 5. Just Fit (ジャスト・フィット)
**タイミングゲーム**
左右に動くバーを、ターゲットゾーンの中央（緑色のエリア）で止めます。
中心に近いほど高得点。全3ラウンドの合計スコアで競います。

## 遊び方

1.  `index.html` をブラウザで開きます。
2.  対戦相手と向かい合って座ります。
3.  メニューから遊びたいゲームを選んで「PLAY」を押します。
4.  それぞれの持ち場（Player 1 は下側、Player 2 は上側）で操作します。

## 開発環境

*   HTML5 / CSS3 / Vanilla JavaScript
*   特別なビルドツールは不要で、そのままブラウザで動作します。
